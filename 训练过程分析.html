计算公式

前向传播

O(n,x,y)=∑c=0C−1∑j=0K−1∑j=0K−1F(n,c,i,j)∗IP(c,yS+j,xs+i)=∑c=0C−1∑j=0K−1∑j=0K−1W(n,c,i,j)∗I(c,yS+j−pt,xs+i−pl)\begin{aligned} O(n,x,y)&= \sum_{c=0}^{C-1}{ \sum_{j=0}^{K-1}{ \sum_{j=0}^{K-1}{ F(n,c,i,j)*IP(c,yS+j,xs+i) } } } \\ &=\sum_{c=0}^{C-1}{ \sum_{j=0}^{K-1}{ \sum_{j=0}^{K-1}{ W(n,c,i,j)*I(c,yS+j-pt,xs+i-pl) } } } \end{aligned}O(n,x,y)​=c=0∑C−1​j=0∑K−1​j=0∑K−1​F(n,c,i,j)∗IP(c,yS+j,xs+i)=c=0∑C−1​j=0∑K−1​j=0∑K−1​W(n,c,i,j)∗I(c,yS+j−pt,xs+i−pl)​
其中，第二行由于去掉了padding，可能发生溢出

反向传播

  • stride = 1
    dLdI(c,y,x)=dLdO∗dOdI(c,y,x)=∑n=0N−1∑j=0H−1∑i=0W−1dLdO(n,j,i)∗dO(n,j,i)dI(c,y,x),其中大部分I(c,y,x)对O没有贡献=∑n=0N−1∑j=0H−1∑i=0W−1dLdO(n,j,i)∗W(n,c,y−j+pt,x−i+pl)令j′=y−j+pt,i′=y−i+pl=∑n=0N−1∑j′=0K−1∑i′=0K−1dLdO(n,y−j′+pt,x−i′+pt)∗W(n,c,j′,i′)令j=K−1−j′,i=K−1−i′=∑n=0N−1∑j=0K−1∑i=0K−1dLdO(n,j+y+(pt−K+1),i+x+(pl−K+1))∗W(n,c,K−1−j,K−1−i)令dLdO的padding为(K−1−pt,HI−HO+pt,K−1−pl,WI−WO+pl),得到dLdOP=∑n=0N−1∑j=0K−1∑i=0K−1dLdOP(n,j+y,i+x)∗W(n,c,K−1−j,K−1−i)\begin{aligned} \frac{dL}{dI(c,y,x)} &=\frac{dL}{dO}*\frac{dO}{dI(c,y,x)} \\ &=\sum_{n=0}^{N-1}{ \sum_{j=0}^{H-1}{ \sum_{i=0}^{W-1}{ \frac{dL}{dO(n,j,i)}*\frac{dO(n,j,i)}{dI(c,y,x)} } } } , 其中大部分I(c,y,x)对O没有贡献 \\ &=\sum_{n=0}^{N-1}{ \sum_{j=0}^{H-1}{ \sum_{i=0}^{W-1}{ \frac{dL}{dO(n,j,i)}*W(n,c,y-j+pt,x-i+pl) } } } \\ & 令j'=y-j+pt,i'=y-i+pl \\ &=\sum_{n=0}^{N-1}{ \sum_{j'=0}^{K-1}{ \sum_{i'=0}^{K-1}{ \frac{dL}{dO(n,y-j'+pt,x-i'+pt)}*W(n,c,j',i') } } } \\ & 令j=K-1-j',i=K-1-i' \\ &=\sum_{n=0}^{N-1}{ \sum_{j=0}^{K-1}{ \sum_{i=0}^{K-1}{ \frac{dL}{dO(n,j+y+(pt-K+1), i+x+(pl-K+1))}*W(n,c,K-1-j,K-1-i) } } } \\ &令\frac{dL}{dO}的padding为( K-1-pt , HI-HO+pt , K-1-pl , WI-WO+pl) , 得到 \frac{dL}{dOP} \\ &=\sum_{n=0}^{N-1}{ \sum_{j=0}^{K-1}{ \sum_{i=0}^{K-1}{ \frac{dL}{dOP(n,j+y, i+x)}*W(n,c,K-1-j,K-1-i) } } } \end{aligned}dI(c,y,x)dL​​=dOdL​∗dI(c,y,x)dO​=n=0∑N−1​j=0∑H−1​i=0∑W−1​dO(n,j,i)dL​∗dI(c,y,x)dO(n,j,i)​,其中大部分I(c,y,x)对O没有贡献=n=0∑N−1​j=0∑H−1​i=0∑W−1​dO(n,j,i)dL​∗W(n,c,y−j+pt,x−i+pl)令j′=y−j+pt,i′=y−i+pl=n=0∑N−1​j′=0∑K−1​i′=0∑K−1​dO(n,y−j′+pt,x−i′+pt)dL​∗W(n,c,j′,i′)令j=K−1−j′,i=K−1−i′=n=0∑N−1​j=0∑K−1​i=0∑K−1​dO(n,j+y+(pt−K+1),i+x+(pl−K+1))dL​∗W(n,c,K−1−j,K−1−i)令dOdL​的padding为(K−1−pt,HI−HO+pt,K−1−pl,WI−WO+pl),得到dOPdL​=n=0∑N−1​j=0∑K−1​i=0∑K−1​dOP(n,j+y,i+x)dL​∗W(n,c,K−1−j,K−1−i)​

dLdI(n)=∑n=0N−1conv(dLdOP(n),WT(n,c))\begin{aligned} \frac{dL}{dI(n)} &=\sum_{n=0}^{N-1}{conv(\frac{dL}{dOP(n)} , W^T(n,c))} \end{aligned}dI(n)dL​​=n=0∑N−1​conv(dOP(n)dL​,WT(n,c))​
即dLdI\frac{dL}{dI}dIdL​是dLdOP\frac{dL}{dOP}dOPdL​与W旋转180°按输出维度的组合卷积并求和

权重更新

  • 对bias梯度
    dLdB(n)=∑hHO−1dO(n,h,w)\begin{aligned} \frac{dL}{dB(n)}&= \sum_{h}^{HO-1}{ dO(n,h,w) } \end{aligned}dB(n)dL​​=h∑HO−1​dO(n,h,w)​

  • 对weight梯度
    dLdW(n,c,y,x)=dLdO∗dOdW(n,c,y,x)=∑j=0H−1∑i=0W−1dLdO(n,j,i)∗dO(n,j,i)dW(n,c,y,x)=∑j=0H−1∑i=0W−1dLdO(n,j,i)∗IP(n,c,y,x)\begin{aligned} \frac{dL}{dW(n,c,y,x)} &=\frac{dL}{dO}*\frac{dO}{dW(n,c,y,x)} \\ &=\sum_{j=0}^{H-1}{ \sum_{i=0}^{W-1}{ \frac{dL}{dO(n,j,i)}*\frac{dO(n,j,i)}{dW(n,c,y,x)} } }\\ &=\sum_{j=0}^{H-1}{ \sum_{i=0}^{W-1}{ \frac{dL}{dO(n,j,i)}*IP(n,c,y,x) } }\\ \end{aligned}dW(n,c,y,x)dL​​=dOdL​∗dW(n,c,y,x)dO​=j=0∑H−1​i=0∑W−1​dO(n,j,i)dL​∗dW(n,c,y,x)dO(n,j,i)​=j=0∑H−1​i=0∑W−1​dO(n,j,i)dL​∗IP(n,c,y,x)​

因此，
dLdW(n,c)=conv(dLdO(n),IP(c))\begin{aligned} \frac{dL}{dW(n,c)} = conv(\frac{dL}{dO(n)} , IP(c) ) \end{aligned}dW(n,c)dL​=conv(dO(n)dL​,IP(c))​
